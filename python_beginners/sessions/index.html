Python For New Programmers

This course aims to give attendees a basic grounding in the concepts of programming (focussing on the Python implementation). It is not going to delve deeply into the specifics of Python, or address specific programming areas such as Database access or programming a Graphical User Interface.

Day 1

In the first day, we're aiming to cover the introductory stuff, and getting used to the platform. We're then going to briefly introduce the essential concepts before gently leading students into writing algorithms - the backbone of programming. We'll end the day with our first real, user interacting program.
