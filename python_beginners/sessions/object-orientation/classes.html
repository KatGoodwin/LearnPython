Simple Classes in Python

What is an object?

  • A 'package' of data and functions
    • e.g. file_object = open('file.txt')
  • ... containing member variables (data)
    • print file_object.name
  • ... and methods (functions)
    • s = file_object.readline()

What is a class?

  • A design (i.e. source code) for an object
  • Defines methods that can be called on the object
  • The methods can define member variables stored in the object

Defining and Instantiating Classes

class MyClass(object):
    def method1(self, arg):
        self.member_variable = arg
ob = MyClass()
ob.method1("some value")
  • MyClass is a class ("design for an object")
  • ob is an instance of that class
  • The code MyClass() constructs the instance
  • Example

The initialiser

class MyClass(object):
    def __init__(self):
        print "hello"
ob = MyClass()
  • __init__ is a special method
  • It is called when an object is created
  • It can take arguments
  • It should not return anything
  • Example

Inheritance

class SubClass(BaseClass):
    # ...
  • Recall that a class is a design for an object
  • A class may inherit from another class
    • The original class is the base class
    • The new subclass extends the original design
  • Examples: a base class and a subclass
