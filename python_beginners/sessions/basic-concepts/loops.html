Loops and Conditions

What is a Loop?

A control construct that allows us to repeat code
  • A specific number of times
  • Until a condition becomes false
    • While the current item in a list isn't the last one
      • While there are more items to process
      • Once for each item in a sequence

Loop structure

<Loop definition>
    code that is run
    each time the 
    loop is executed
    ....
